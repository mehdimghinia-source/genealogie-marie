## Stratégie de validation (Chrome MCP)

### Étape 1 — Acte naissance 21/12/1906 Cruet sur AD73

1. Naviguer https://recherche-archives.savoie.fr/
2. Recherche guidée → Commune **Cruet** → type **Naissances** → période **1906**
3. Particularité : pour les petites communes savoyardes, **tables décennales 1903-1912 souvent reliées en fin de registre annuel** plutôt que dans un registre séparé → feuilleter directement le registre 1906
4. Trouver l'acte du **21 décembre 1906** (probablement parmi les derniers actes de l'année)
5. **À extraire** :
   - Numéro d'acte
   - Heure de naissance
   - Père : nom + prénom + âge + profession + lieu de naissance + village résidence Cruet
   - Mère : nom de jeune fille + prénom + âge + profession
   - Témoins
   - Signatures (lettré ou non)
   - Mention marginale (généralement : mariage 11/08/1928 Paris 7e, décès si trouvé)

### Étape 2 — Acte mariage 11/08/1928 Paris 7e

1. Naviguer https://archives.paris.fr/r/42/etat-civil-parisien-a-partir-de-1860-/
2. État civil → 7e arrondissement → tables décennales mariages 1923-1932 → lettre M (MONIN) → cote
3. Registre annuel mariages 7e 1928 → acte du 11/08/1928 ou environs
4. **À extraire** :
   - Profession des époux à 1928 (Amédée 22 ans = ouvrier ? employé ? militaire ?)
