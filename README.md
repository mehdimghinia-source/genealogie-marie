# Arbre généalogique — Famille Marie PIERONNE

Site privé protégé par mot de passe (partagé en famille uniquement).

🔐 Le contenu est chiffré côté client (AES via [staticrypt](https://github.com/robinmoisson/staticrypt)). Aucun serveur n'est requis — GitHub Pages sert juste les fichiers chiffrés statiques. Sans le mot de passe, le contenu reste illisible.
